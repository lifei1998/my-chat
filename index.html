Core Capabilities

Chat ဆက်သွယ်ရေးကို ရိုးရှင်းအောင် ပြန်လည်ဒီဇိုင်းဆွဲထားပါတယ်

01

Unified Inbox

Platform အသီးသီးက message တွေကို တစ်နေရာတည်းမှာ စုစည်းပြီး စီမံနိုင်ပါတယ်။

02

Instant Translation

ပို့တာနှင့် လက်ခံရတာ message နှစ်ဖက်စလုံးကို 0.3 စက္ကန့်အတွင်း ဘာသာပြန်ပေးပါတယ်။

03

Voice & Image

Voice message နှင့် ပုံထဲက စာသားကိုပါ ဘာသာပြန်နိုင်ပါတယ်။

04

End-to-End Privacy

Translation cache တွေကို device ပေါ်မှာပဲ သိမ်းဆည်းပြီး cloud ကို မပို့ပါ။

How it works

သုံးရလွယ်ကူပါတယ်၊ ၃ ဆင့်ပဲ

01

App ကို install လုပ်ပါ

Windows သို့မဟုတ် macOS အတွက် installer ကို ဒေါင်းလုဒ်ဆွဲပြီး run လုပ်ရုံပါပဲ။

02

Account တွေချိတ်ဆက်ပါ

WhatsApp, Telegram, Messenger စာရင်းများကို QR scan ဖြင့် ချိတ်ဆက်ပါ။

03

ဘာသာစကားရွေးပါ

ကြိုက်နှစ်သက်ရာ ဘာသာစကားကို သတ်မှတ်လိုက်ရင် ချက်ချင်း ဘာသာပြန်စပါလိမ့်မယ်။
